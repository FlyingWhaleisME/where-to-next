## **TOOL 1: Database (MongoDB + Mongoose)**

### **Initial Research & Learning**

**Problem:** "How do I store data permanently in a web application?"

**Search Queries Used:**
- "how to store data in node.js web application"
- "MongoDB vs SQL database for beginners"
- "MongoDB tutorial node.js"
- "Mongoose tutorial express"

### **Learning Resources:**

#### **Videos:**
1. **"MongoDB Crash Course" by Traversy Media**
   - YouTube: https://www.youtube.com/watch?v=ofme2o29ngU
   - **Why:** Comprehensive introduction to MongoDB concepts, perfect for beginners
   - **What I learned:** Basic MongoDB operations, document structure, NoSQL concepts

2. **"Mongoose Tutorial for Beginners" by Net Ninja**
   - YouTube: https://www.youtube.com/watch?v=DZBGEVgL2eE
   - **Why:** Step-by-step Mongoose tutorial specifically for Node.js/Express
   - **What I learned:** Schema definition, model creation, CRUD operations with Mongoose

#### **Blogs & Documentation:**
1. **MongoDB Official Documentation - Getting Started**
   - URL: https://www.mongodb.com/docs/manual/tutorial/getting-started/
   - **Why:** Official documentation is always the most reliable source
   - **What I learned:** MongoDB fundamentals, installation, basic operations

2. **Mongoose Official Guide**
   - URL: https://mongoosejs.com/docs/guide.html
   - **Why:** Complete reference for Mongoose features and best practices
   - **What I learned:** Schema definition, models, middleware, validation

3. **"MongoDB vs MySQL: Which Database Should You Use?" - FreeCodeCamp**
   - URL: https://www.freecodecamp.org/news/mongodb-vs-mysql/
   - **Why:** Helped me understand when to use MongoDB vs SQL databases
   - **What I learned:** Differences between NoSQL and SQL, use cases for each

4. **"Async/Await in Node.js" - Node.js Official Docs**
   - URL: https://nodejs.org/en/learn/asynchronous-work/async-await
   - **Why:** Essential for understanding async database operations
   - **What I learned:** How async/await works with database queries

### **Implementation Help:**
- **Stack Overflow:** Searched "mongoose save document example", "mongoose findOne async await"
- **GitHub:** Looked at example projects using Mongoose with Express

I started with Traversy Media's 'MongoDB Crash Course' video tutorial to understand NoSQL database concepts, then followed Net Ninja's 'Mongoose Tutorial for Beginners' to learn how to integrate MongoDB with Node.js. The Mongoose official documentation provided detailed information about schema definition and validation. This research helped me understand why MongoDB's flexible document structure was suitable for storing varying trip data structures compared to rigid SQL table schemas."

### **What to Include:**
- ✅ **Initial problem/question** you were trying to solve
- ✅ **Resources consulted** (videos, blogs, documentation)
- ✅ **What you learned** from each resource
- ✅ **How it helped** you make implementation decisions
- ✅ **Why you chose** that tool over alternatives (based on research)

- ❌ Don't list every single resource (pick 2-3 main ones per tool)
